업체별 화물 정보 취합
▼

업체마다 아래 정보를 반드시 확보해야 합니다.

  • 팔렛 수 확인
    몇 팔렛인지 정확히 확정 (예: 4PLT, 2PLT)
  • 팔렛별 사이즈 (W × L × H, mm 단위)
    골드웨이 기준: 1100×1100 기본, H는 업체마다 상이
  • CBM 계산 확인
    W×L×H(m) = CBM. 팔렛 수 × 개별 CBM 합산
  • G.W (총 중량, kg) 확인
    팔렛 포함 실중량 기준
  • 출고 가능일 확정
    CFS 마감일보다 여유 있는지 반드시 확인
  • 화물 종류 확인 (일반/위험물/냉동)
    위험물은 UN No., IMO Class 별도 확인 필요

픽업 & 배차 정보 확인
▼
  • 픽업 주소 확보 (업체별)
    예: 경기도 광주시 오포읍 오포로 650-23 오포CS물류센터 3F
  • 담당자 연락처 확인
    픽업 당일 연락 가능한 현장 담당자 번호
  • 픽업 가능 시간대 확인
    점심시간(13~14시) 등 제한 시간 있는지 확인
  • 합차 픽업 여부 확인
    여러 업체 동선 묶어서 픽업 가능한지
  • 배차 정보 업체에 공유
    차량 번호, 기사 연락처 픽업 전날 발송
  • 도착 보고 요청
    CFS 입고 완료 후 업체에 입고 확인 공유
